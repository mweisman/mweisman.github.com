Puppet

Single Cross-platform deploy script

(even works on Windows with a bit of futzing!)
Did you know there is an OpenGeo Suite puppet module?

Vagrant

Puppet setup:

class {'opengeosuite': repo => 'test', version => '3'}

Vangrant setup:

config.vm.provision :puppet do |puppet| puppet.module_path = "../../puppet_modules" puppet.manifests_path = "." puppet.manifest_file = "apply.pp"
